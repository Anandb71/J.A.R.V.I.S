# Support

## Getting help

For usage questions and non-security bugs, please open a GitHub issue.

## Before opening an issue

- Confirm you are on the latest `main`
- Include OS, Node, and Python versions
- Include steps to reproduce and logs

## Security issues

For vulnerabilities, follow `SECURITY.md` and use a private security advisory.

## Scope note

This project is currently optimized for Windows desktop environments.
