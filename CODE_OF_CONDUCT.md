# Code of Conduct

## Our Pledge

We are committed to a welcoming, harassment-free community for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Respectful and constructive communication
- Helpful feedback focused on code and behavior
- Openness to different experiences and skill levels

Examples of unacceptable behavior:

- Harassment, intimidation, or discrimination
- Personal attacks, trolling, or insulting language
- Publishing private information without consent

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing this code of conduct.

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, and discussions.

## Enforcement

Report behavior that violates this code to project maintainers through private channels listed in `SECURITY.md`.
